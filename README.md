# Retail Sales Data Analysis

## Project Overview
This project analyzes retail transaction data to uncover insights into sales performance, profitability, discount strategies, and regional trends.

## Objectives
- Identify top-performing and underperforming product categories
- Analyze the impact of discounts on profit
- Evaluate regional and city-level sales performance
- Detect loss-making products

## Tools & Technologies
- Python (Pandas, Matplotlib)
- Excel
- Power Point

## Key Findings
- High discounts increased sales volume but reduced overall profitability
- Certain regions consistently underperformed and require targeted strategies
- A small number of products contributed disproportionately to losses

## Recommendations
1. Refine Discount Strategy
2. Strengthen Underperforming Regions
3. Focus on High Value Cities
4. Product Level Optimization
5. Seasonal Marketing Strategy
6. Payment Method Optimization
7. Improve Profit Margins
